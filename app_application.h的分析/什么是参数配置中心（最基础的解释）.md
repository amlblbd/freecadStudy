## 什么是参数配置中心（最基础的解释）

想象参数配置中心就像一个**大文件夹柜**，里面有很多**抽屉**，每个抽屉里有很多**小纸条**，上面写着各种设置。

### 1. 最基础的概念

**参数配置中心 = 一个存放设置的地方**

就像你电脑上的"设置"或"偏好设置"一样，FreeCAD把各种设置都存在这里。

### 2. 三个基本操作

```python
# 1. 读取设置（看看纸条上写了什么）
value = FreeCAD.ParamGet("路径").GetBool("设置名", 默认值)

# 2. 保存设置（在纸条上写东西）
FreeCAD.ParamGet("路径").SetBool("设置名", 新值)

# 3. 删除设置（把纸条扔掉）
FreeCAD.ParamGet("路径").RemBool("设置名")
```

### 3. 四种数据类型（四种不同颜色的纸条）

```python
# 布尔值（是/否）- 就像开关
FreeCAD.ParamGet("路径").GetBool("显示网格", True)  # 读取
FreeCAD.ParamGet("路径").SetBool("显示网格", False)  # 设置

# 整数（整数）- 就像计数器
FreeCAD.ParamGet("路径").GetInt("网格大小", 10)  # 读取
FreeCAD.ParamGet("路径").SetInt("网格大小", 20)  # 设置

# 浮点数（小数）- 就像尺子
FreeCAD.ParamGet("路径").GetFloat("线宽", 1.5)  # 读取
FreeCAD.ParamGet("路径").SetFloat("线宽", 2.0)  # 设置

# 字符串（文字）- 就像便签
FreeCAD.ParamGet("路径").GetString("用户名", "默认")  # 读取
FreeCAD.ParamGet("路径").SetString("用户名", "张三")  # 设置
```

### 4. 路径是什么？（文件夹结构）

路径就像文件夹路径，用斜杠分隔：

```
"User parameter:BaseApp/Preferences/Mod/Draft"
```

这表示：
- `User parameter` - 用户设置（还有 System parameter 系统设置）
- `BaseApp` - 基础应用
- `Preferences` - 偏好设置
- `Mod/Draft` - Draft模块的设置

### 5. 实际例子（来自真实代码）

```python
# 例子1：读取是否精炼模型（来自Part模块）
def getParamRefine():
    # 读取"精炼模型"这个设置，默认为False
    return FreeCAD.ParamGet("User parameter:BaseApp/Preferences/Mod/Part/Boolean").GetBool("RefineModel")

# 例子2：保存网格设置（来自BIM模块）
# 保存网格间距为"1 cm"
FreeCAD.ParamGet("User parameter:BaseApp/Preferences/Mod/Draft").SetString("gridSpacing", "1 cm")

# 保存网格数量为10
FreeCAD.ParamGet("User parameter:BaseApp/Preferences/Mod/Draft").SetInt("gridEvery", 10)
```

### 6. 为什么要用参数配置中心？

**想象场景：**
- 你关闭了FreeCAD，下次打开时还想用同样的设置
- 不同模块需要共享一些设置
- 用户可以自定义各种行为
- 程序需要记住用户的偏好

### 7. 最简单的记忆方法

```
ParamGet("路径") → 找到抽屉
.GetBool() / .SetBool() → 看/写开关纸条
.GetInt() / .SetInt() → 看/写数字纸条  
.GetFloat() / .SetFloat() → 看/写小数纸条
.GetString() / .SetString() → 看/写文字纸条
```

### 8. 参数配置中心 = 程序的记忆

就像人的大脑记住偏好一样，参数配置中心让FreeCAD能够：
- 记住你的设置
- 下次打开时还是你喜欢的样子
- 不同功能模块可以共享设置
- 用户可以自定义程序行为
